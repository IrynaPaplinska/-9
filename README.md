Опис алгоритму:
Програма зчитує з клавіатури дані про вісім потягів, збережених у структурі Train. Далі вона впорядковує цей масив за часом відправлення методом порівняння і перестановки елементів. Після цього програма виводить на екран інформацію про всі потяги, що прямують до введеного користувачем міста, або повідомляє про їх відсутність.
